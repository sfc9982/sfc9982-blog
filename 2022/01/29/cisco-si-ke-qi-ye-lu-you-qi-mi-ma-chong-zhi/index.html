Cisco思科企业路由器密码重置

  1. 此时如果可以通过调试线访问路由器的话，检查是否有show version命令权限。输出的最后一行提示了设置文件寄存器的地址。如果没有种种原因，无法访问路由器，那么此时设置文件寄存器的地址应为0x2102。 

    Router>show version
<output omitted>
Configuration register is 0x2102

  2. 重启路由器（reload），并在启动后的60s内中断启动，以进入ROMMON模式。中断方式见附录。

  3. 在rommon 1>后输入confreg 0x2142以从闪存启动。此条命令会使路由器在下次重置时跳过启动设置步骤。 

    rommon 1 > confreg 0x2142
You must reset or power cycle for new config to take effect

  4. 在rommon 2>后输入reset 以重启路由器。 

    rommon 2 > reset
System Bootstrap, Version 11.3(2)XA4, RELEASE SOFTWARE (fc1)
Copyright (c) 1999 by cisco Systems, Inc.
TAC:Home:SW:IOS:Specials for info
C2600 platform with 32768 Kbytes of main memory
<output Omitted>
router>

  5. 在设置向导质询中回答no或者按Ctrl-C以跳过初始化设置向导。 

    Would you like to enter the initial configuration dialog? [yes/no]: n

  6. 输入enable进入Enable模式 

    Router>enable
Router#

  7. 如果要恢复密码，输入configure memory或copy startup-config running-config。 

    Router#copy startup-config running-config

  8. 输入show running-config 以确定活动设置文件已被更改。

  9. 输入configure terminal。 

    Router#configure termial

  10. 如要更改密码，输入enable secret <password>，使用hostname(config)# 提示符。此时，可以重置路由器的其他密码。 

    Router# conf t
Router(config)#enable secret <new_secret_password>
Router(config)#enable password <new_enable_password>
Router(config)#line vty 0 15
Router(config-line)#password <new_vty_password>
Router(config-line)#login
Router(config-line)#line con 0
Router(config-line)#password <new_console_password>
Router(config)#^z

  11. 在计划使用的各个Interface中执行no shutdown 命令。

  12. 执行write memory或copy running-config startup-config来将配置文件写入启动过程。 

    Router#copy running-config startup-config

  13. 输入configure termial，返回到全局配置模式。 

    Router#configure terminal

  14. 输入config-register 0x2102并按Ctrl-z以保存更改并退出privilege EXEC模式。 

    Router(config)#config-register 0x2102
Router(config)#^Z

  15. 输入reload重启路由器，此时路由器使用0x2102配置寄存器启动。 

    Router#reload
Your passwords have now been reset.
